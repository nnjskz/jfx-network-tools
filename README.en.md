# jfx-network-tools

#### Description
jfx-network-tools 是一个基于 JavaFX 的轻量级网络调试工具，支持 TCP/UDP 客户端与服务端模式，具备自动应答、连接历史、日志保存等实用功能。界面简洁，操作直观，适合网络协议调试、教育演示与快速开发使用。
